The phone field module contains a directive which validates phone numbers against the standards for different countries. The value that is validated against is driven through the ngModel attribute specified on the input element or a FormControl within a reactive form.

Creates the phone field input and dropdown. Place this attribute on an input element, and wrap the input in a sky-phone-field component. The attribute must be set to the instance of the sky-phone-field component. Specifies the default country for the phone field input and dropdown. This property accepts string International Organization for Standardization Alpha 2 values. Indicates whether to disable the phone field. This property accepts boolean values. Indicates whether to format the phone field's model value based on national formatting rules. This property accepts boolean values.
