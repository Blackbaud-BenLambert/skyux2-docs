SKY UX is more than a set of user interface components. Equally important for a consistent experience, it includes guidelines to describe how to apply components and the context for when to use them.

Page layout

Pages should be designed to match the usage you are designing for. While there are some common layout patterns, a single page template will not work for all scenarios.

Overview A high-level overview of the page structure and common page layouts used in SKY UX applications. Grid system Lay pages out using the SKY UX fluid grid ensure consistent alignment and responsiveness. Responsive design Design pages that work well regardless of screen size. Record page Present the information related to a record in a re-arrangeable tile dashboard. List page Search, filter, sort and take action on a list of records. Workcenter page Provide quick access to commonly-needed information and actions based on role or task. Split view page Work though a list of records to take a specific action on each record.
